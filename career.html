Career Guidance Using ML

About the Project

  • Project Overview: Developed an application that helps freshers choose a suitable IT domain by testing their knowledge.
  • Initially built using the Kivy framework, now upgrading it to the Django framework to enhance scalability and web integration.

  • Dataset Collection:Collected and preprocessed data from Kaggle, ensuring clean and structured input for model training.
  • Developed a machine learning model using the Decision Tree Classifier algorithm to predict the ideal IT domain based on user input.
  • Model Development: Model Integration:Serialized the trained model using the Pickle library and integrated the .pkl file into the Python script for real-time execution of predictions.
